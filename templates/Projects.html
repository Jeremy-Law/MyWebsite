Priority Queues

The second project was my first real dive into what a pointer is. Before this class I did not know or care what a pointer is, but now
I understand what it is and some important uses thanks to this project and others. In case you couldn't tell by the name of the
project, we built a basic priority queue. It had some basic functions like enqueue, dequeue, and insert some basic structures in a linked list using pointers and pointer math.

Dynamic Memory Allocator

Skipping ahead, my favorite project across all the courses I've taken is a basic dynamic memory allocator. For those who may not know what that is,
a dynamic memory allocator will ask for memory from the operating system and distribute it to a program as needed. Ours was not advanced and should not be
used on a large system as it never gave any memory back to the OS for efficiency reasons. But by the time the project was due, we could use it to run some
basic system commands such as printing files to the terminal.
I got a 100% on this project and found it very satisfying when I could run commands and they would work correctly.
